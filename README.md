# Batch Web Exporter — Photoshop Plugin

A UXP panel plugin for Adobe Photoshop that batch exports all layers in the active document as web-ready image files (PNG, JPG, SVG, or WebP).

## Features

- Export all layers or visible layers only
- Choose output format: PNG, JPG, SVG, WebP
- Choose export scale: 1x, 2x, or 3x
- Saves files to a folder you pick

## Requirements

- Adobe Photoshop 23.0 or later (UXP support required)

## Installation

1. Open Photoshop
2. Go to **Plugins → UXP Developer Tool**
3. Click **Add Plugin** and select this folder
4. Click **Load** to activate the panel

## Usage

1. Open a document in Photoshop
2. Open the **Batch Web Exporter** panel from the Plugins menu
3. Choose your format and scale
4. Click **Export All Layers**
5. Select a destination folder when prompted
